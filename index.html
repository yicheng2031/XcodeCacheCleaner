Built for everyday Xcode cleanup

The app keeps cleanup visible and selective. You can inspect categories, expand item lists, and remove only the data you actually want to clean.

Selective by default

Choose full categories or expand Runtimes, Archives, DerivedData, SwiftPM caches, logs, and unavailable simulators.

Runtime aware

Simulator Runtime cleanup uses stable runtime identifiers, so older iOS/watchOS/tvOS/visionOS runtimes can be removed reliably.

Local and private

No account, no tracking, no analytics, no cloud sync. Scanning and deletion run locally on your Mac.

Cleanup coverage / 清理范围

Common Xcode and Apple development cache locations are grouped into clear categories, with destructive items shown before cleanup.

  • DerivedData / 编译产物与索引
